如何评价ry(Ryan Dahl)的新项目deno?
Yorkie
正式回答一下，我觉得 deno 并不会让大多数 node 或者前端程序员失去工作，或者说大家仍然可以继续愉快地使用 node 写代码，拿着你现在一分不少的工资。
因为 deno 本身就不是为了替代 node 或者 npm 存在的，他们解决的就不是一个问题。在服务器开发领域，Node.js 可能更多扮演的是类似 PHP，Ruby，J2EE 这样的角色，他提供的是一个完整的运行时，以及开发领域。
我认真读过 deno 的文档以及少量代码后，认为 deno 更多是首先作为一个沙箱，去运行非安全代码，简单的说就是用户的代码，然后这个环境与现代浏览器兼容，并且允许你去定制你所需要提供的接口。注意，README 中明确说明了没有网络接口。
这种定义下，有以下几种场景我想可以实现。
比如很多基于 golang 实现的数据库，分布式服务甚至 docker，都可以基于 deno 去提供一个基于 v8 的运行时。
另一个非常适合 deno 的场景就是基于 JS 的 serverless 云服务，原来使用 Node.js 要想尽办法封闭的接口，现在好了，我们提供什么，都可以指定，另外基于 proto 和 golang 的底层通讯，也完全可以让服务本身足够健壮和稳定。
另外值得一提的是，deno 中 golang 与 v8 并没有用我们熟知的 ffi 或者序列化接口，取而代之是 protobuffer，也就是说 deno 完全具备分布式部署的能力。
所以看完这些，deno 完全是针对目前 Node.js 并不擅长的领域，做的又一次尝试，如果成功，JavaScript 必然又一次会进行扩展。所以大家完全没必要做类似今天这样的行为。
即便如我这般不喜欢 golang 和 ts 语法的人，今天都耐下心读了源码并且还提了一个 PR，又何况是你呢。
另外，补充一点，关于生态问题，我非常佩服或者说欣赏 ry 的选择，我们大多数人在做类 Node.js 运行时的同时都陷入了 NPM/Node.js 的泥垢中，其实拥抱浏览器才是拥抱真正的生态，ry 作为 Node.js 创始人能这么想，更加值得敬佩！
当然 deno 任然不会是 JavaScript 在设备端/嵌入式平台的运行方案，所以我任然坚信 ShadowNode 的未来并以 ry 为榜样继续努力！

你吃过最令人感动的方便食品或小吃是什么？
苏阐伍
方便面简直是对我这种人最伟大的发明了，我对它充满了敬意，简直拯救了我干枯的人生。
在家，一个人；工作，去边远的地方出差，一个人。形影单只，偶尔想出去吃点好的，但实在挣得少，想想作罢。所以煮泡面简直成为我人生的标杆。
为了让生活不那么无聊和单调，我会闲暇之余去超市买菜。晚上9点以后，超市总会有一大堆打折的瓜果蔬菜。像昨天，2块买了一大袋小西红柿，1块买了四把小青菜。家里还有泡面和香肠，色彩丰富，搭配起来煮一碗真的经济营养也好吃。这样哪怕是泡面，也会变得美味起来。
烧开水，先把调味包用开水冲开，再熟练的洗干净小青菜，把小番茄、香肠切片，发现了一枚卤蛋，切了一半也切成小片。煮面，捞面，过冰水，面更劲道。
小心翼翼端上桌，房里氤氲着香气，擦了擦手，拍照留念，想发朋友圈，应该也没有人在乎我吃泡面吧…就在手机里一直放着，劳累一整天，吃一顿面从头放松到胃。




有时，我妈打电话会问，“最近吃的好嘛？！是不是又吃方便面？不准吃泡面，有防腐剂的噻，还没营养。你在外头子要好好吃，身体第一位知道吗，我给你讲，上次我看有人讲…”
“我早都不吃泡面了妈，真的，现在吃的很有营养。等我有时间我就回家去看你。”
我妈也很满意的挂了电话。
那能怎么样呢，哈哈。我收好手机，打了两个鸡蛋，鸡蛋轻轻荡漾在锅里，逐渐散出荷包蛋的形状，可能是我火开大了，总是会把蛋清煮散。
我妈煮蛋煮的很好啊，记得在家的时候，我早上刚醒去洗漱，她一边摆碗筷一边笑着说
“崽儿啊，这就起了，不多睡会？我给打了两个蛋，你看着自己放糖。这有小咸菜。”
我接过刚出锅的荷包蛋，放一勺白糖。就着咸菜，很满足了。
啊水开了，收回来思绪。迅速切了点胡萝卜丁和小葱花，过水烫熟，想了想，切了点本来打算做三明治的午餐肉，也烫了一下。面煮熟，没过冰水，今天外头下雨，有些冷。
大功告成。


这样的日子不知还有多久，虽然难了些，但晚上听着雨，吸溜点面，还能再咬个蛋吃，也没那么冷了。
生活，哪有什么最令人感动，每天拼命的活着，都令人感恩啊。

知乎用户
1923年至1940年，美国海军进行了21次“舰队问题”演习。

第一次“舰队问题”演习，即Fleet problem I 就实验了舰载航空兵对岸基设施的打击能力。

演习内容是防御巴拿马运河，由“俄克拉荷马”号战列舰扮演的航空母舰，起飞了一架舰载机，成功摧毁了巴拿马运河设施。


“舰队问题III”演习依旧以巴拿马运河为舞台，测试了两栖登陆技术。


1925年“舰队问题V”模拟了袭击珍珠港，扮演进攻角色的黑方首次投入了航空母舰“兰利”号。

而此次演习中“兰利”号的活跃表现大力推进了随后“列克星敦”级航空母舰的改建工程和战斗力形成。


1927年，“舰队问题VII”中“兰利”号又一次成功空袭了巴拿马运河。


1928年的“舰队问题VIII”则演练了反潜作战。


随后的“舰队问题IX”中“列克星敦”号和她妹妹“萨拉托加”号航母分属攻防双方，扮演攻击方的“萨拉托加”又一次把巴拿马运河给炸了，证实了以航母为核心的快速打击舰队在作战能力上的多样性和有效性。


“舰队问题X”里，“萨拉托加”一举击沉了“兰利”和姐姐“列克星敦”，迅速改变了演习双方实力对比。

第一，军人不要居功自傲。过去的荣誉永远是属于过去的，不是护身符，不会永远保着我们，时运不济时可能也会草草结束军旅生涯，而到了地方这些荣誉除了吹吹牛逼之外对我们毫无帮助，国家现在也只能给我们这些了，以后会越来越好，与其现在抱怨，不如靠自己拼搏和努力创造新生活。
第二，也不要一直把英雄的光环套在军人身上。军人退役后，就是和其他人一样的普通人，也要为生活奔波，为家人打拼，也会变成很普通很普通，不能再普通的人，可能就是身边的快递员，超市的小老板，上下班的公务员。英雄即是一种荣誉，也是一种压力，你们也要接受活着的英雄最终会变成跟你们一样的普通人这个现实，军人退役后也要回归社会回归家庭，也要卸下身上担子，活着的英雄不可能成为永远的英雄，他们不可能永远在那个位置不下来，这意味他们要一直做出丰功伟绩和巨大付出，谁也不可能永远这样，只有在成为英雄那一瞬间牺牲的人才能不朽，永远保持英雄的样子不会再变成普通人。




再次补充，我就很奇怪，总有人觉得军人立下了丰功伟绩，国家就应该供养一辈子，我们的一等功臣退役后，过的不理想就应该找国家要这要那。抱歉，军人有军人的尊严，荣誉和物质奖励是为了激励我们建功立业的，不是要把我们圈养起来的，真的因公因战致残，让国家养一辈子没问题，现在这方面的保障也比以前完善了很多，只要经过伤病残鉴定，被鉴定为一到六级伤残或者丧失工作能力的，都是可以病退，真的是国家供养一辈子。其他达不到病退等级的，每年都可以领伤残抚恤金，退役后由民政部门给发，不会再出现丧失工作能力的残疾军人老无所依的情况。还有的战友说，对干部保障好，对战士的保障还不够，但是2011出台的《退役士兵安置条例》也在完善这一点，只要是荣立平时二等功、战时三等功以上奖励或服役满12年的士兵都是给安排工作的，按照这个标准，题目里这些空降兵士官都是可以由国家给安排工作的，遗憾他们没有赶上这些政策，毕竟制度的完善是要随着国家经济发展和国力强大逐步进行的。我知道我不是英雄，我没有资格替这些英雄说话，下面我只讲讲个人感受，我刚到部队时，一个月工资2000多，到转业最高达到9800多(包括边海防补助和艰苦边远补助，边海防补助武警也有，只要是在边防海防线上)；2008年因为奥运安保立了个三等功，奖励费只有200块钱，等到2014年再立三等功时已经涨到2000块钱翻了10倍；转业时，国家安排了工作，还给发了34万转业费(有11万是2014年10月之后，全军统一开始交社保，部队给交的社保和职业年金，转业后要存进个人的社保账户，不发给个人)，到手23万，领到了国防服役铜质纪念章(服役满10年)和卫国戍边铜质纪念章(在边境地区服役满一定年限)，献身国防纪念章没有领到(只有因战因公致残牺牲的才能领)。福利待遇提升的同时也感受到了军人社会地位的提升，转业离队后因为很多空闲的时间，带着媳妇儿去杭州玩了一趟，回来时在萧山机场因为军人依法优先走的绿色通道先安检先登机(当时还没有正式办转业相关手续，武警警官证还没有上交)，在岳王庙、千岛湖和横店影视城都免了门票，媳妇儿感到非常的幸福，但是让我感到惭愧和遗憾的是，在部队这么多年只让她感受过一次这种待遇，还是在要离开部队时才有了大把时间的情况下。说这些，我只想说虽然现在的福利待遇还不能完全配得上军人的付出，就拿我个人来说，军旅生涯基本没有回家过过完整的春节，甚至还有很多次一天也没有回去的，2008年汶川地震加上奥运安保，全年没回过一天家，但是总体上看这些年军人的福利待遇还是在飞速提升的，我来回答这个问题，只是想表达这些年我对国家、对人民给予我的这些待遇心里是充满感激和愧疚的，没有为这个国家贡献更多，也没有成为评论区有些人嘴里的英雄，更没有替这些英雄们说话的资格，我只能凭我低劣卑微的道德水准和精神境界斗胆猜测一下，这些英雄模范当年站在飞机上面对生死纵身一跃，肯定不是为了跳下来跟国家要待遇要福利的，即使在退役时这个国家没有给他们应有的待遇，他们应该也是把这一跳当成这一生最荣耀最辉煌的一刻，而不是拿它当资本到处炫耀和索要待遇的，以上瑾代表我个人，要是达不到评论区里一些人的期待，实在抱歉了。
